# 服务方式

早期阶段以真实用户试用、问题诊断和工作流打磨为主。价格和范围会根据场景人工确认。

## 单次诊断

适合已经有明确 Marc/Mentat 问题的用户，例如：

- solver exit number；
- 收敛问题；
- 非正定；
- 接触、连接或边界条件可疑；
- 后处理提取困难。

可能交付：

- 简短诊断摘要；
- 日志或截图中的证据线索；
- 建议继续检查的方向；
- 必要时给出操作建议。

## 月度使用

适合论文、科研项目或工程流程中需要反复使用的用户。

可能包括：

- 闭源试用/Pro 包授权；
- 有效期内的有限支持；
- 版本更新说明；
- 工作流建议。

月度使用是比较推荐的模式，因为它能支撑持续迭代和真实问题反馈。

## 项目支持

适合需要更深度整理 Marc 工作流的场景。

可能包括：

- 定制化工作流设计；
- 模型检查流程；
- 求解诊断流程；
- 结果提取和报告流程；
- 交接说明。

## 边界

本工具辅助 Marc/Mentat 工作流，不替代工程判断、模型验证或有效 MSC Marc/Mentat 授权。付款前应先确认支持范围。
