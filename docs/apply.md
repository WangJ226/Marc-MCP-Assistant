## Best-Fit Trial Cases

The current private package is most suitable when you want to:

- connect Codex to a local MSC Marc/Mentat workflow through a controlled MCP
  server;
- check whether your local Marc/Mentat environment can be detected correctly;
- use Codex to help organize model-inspection clues, solver-failure clues, and
  selected post-processing evidence;
- make your Marc workflow more traceable for research, thesis, or engineering
  projects.

It is less suitable if you only want a public open-source MCP example, an
unlimited free tool, or a replacement for Marc/Mentat engineering judgment.

## Please Do Not Send First

Please do not send these in the first public contact:

- full model files;
- confidential company, lab, or project data;
- unredacted local paths, customer names, contract details, or server details;
- Marc/Mentat license files.

If later diagnosis requires more information, screenshots, short log excerpts,
or anonymized error messages are preferred first. Full model files should only be
discussed after explicit agreement.

## What You May Receive

If your use case fits the current private beta, you may receive:

- Windows closed-source trial package;

- short setup instructions;

- feedback template.

The first guided check usually confirms:

```text
package status
license status

```

## Typical Trial Flow

```text
Read this GitHub page
-> contact the author
-> describe your Marc/Mentat version and current problem
-> receive the private Windows package if suitable
-> run status check and send machine code
-> receive license.json
-> configure Codex MCP with the setup prompt
-> verify the executable path
-> test on backup or non-critical models
-> send concise feedback if something fails
```

## Trial Boundary

Trial licenses may be limited by time, machine, feature set, package version, or
support scope. The goal is to verify whether the assistant helps your real Marc
workflow.

The trial package is not for redistribution, resale, reverse engineering, or
building a competing package from the delivered files.
